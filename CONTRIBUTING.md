# Contributing to Awesome AI Resources

Thank you for your interest in contributing to this collection! This document provides guidelines for contributions.

## How to Contribute

1. **Fork the repository**
2. **Create a new branch** for your changes
3. **Make your changes** following the guidelines below
4. **Submit a pull request** with a clear description

## Guidelines

### Adding New Resources

- Ensure the resource is relevant to AI/ML
- Check that it's not already listed
- Add it to the appropriate category
- Include a brief description
- Verify all links are working

### Format

```markdown
- **[Resource Name](URL)** - Brief description of what it is and why it's useful
```

### Categories

If you think a new category is needed, please open an issue first to discuss it.

## Code of Conduct

- Be respectful and inclusive
- Focus on constructive feedback
- Help maintain a positive community

Thank you for helping make this resource better!
